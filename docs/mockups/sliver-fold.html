The model: folding a block substitutes its content for a digest — that digest is the cocoa block, a real (synthetic) block that genuinely occupies the context window, so it gets a dice face for its own (small) size. The sliver beside it is the original, kept visible thin so you still read its weight. Grouping is the only thing that makes blocks share one cocoa summary — adjacency alone never does. So an ungrouped fold is a 1-cocoa + 1-sliver pair; an explicit group is 1-cocoa + N-slivers. Tradeoff to weigh: each ungrouped fold is two tiles (cocoa + sliver) rather than one, so a folded region isn't more compact than today — the win is legibility (you read the digest, and the original's weight, at a glance), not density.
